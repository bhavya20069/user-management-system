# User Management System

## Project Description
This project is a User Management System developed using PHP and MySQL.

It includes:
- User Registration
- User Login
- Dashboard
- Users Table
- Add User
- Edit User
- Delete User
- Profile Page
- Image Upload

---

## Technologies Used

- PHP
- MySQL
- HTML
- XAMPP
- phpMyAdmin

---

## Features

### Authentication
- User Registration
- User Login
- Session Management
- Logout

### CRUD Operations
- Create User
- Read User Data
- Update User Data
- Delete User Data

### Profile Management
- Upload Profile Image
- Display Username
- Display Email

## How to Run

1. Start Apache and MySQL in XAMPP
2. Open phpMyAdmin
3. Create database:
   user_management
4. Import/Create users table
5. Place project inside:
   C:\xampp\htdocs\
6. Open browser:

```text
http://localhost/user-management-system
```
